# PageJs

PHP compiler to JavaScript.

## Installation

Via npm on Node:

```
npm install pagejs
```


## Usage

Reference in your program:

```js
var sg = require('pagejs');
```

## Development

```
git clone git://github.com/ajlopez/PageJs.git
cd PageJs
npm install

```
